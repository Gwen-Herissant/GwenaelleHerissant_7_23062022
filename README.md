# Les Petits Plats — JavaScript Search Algorithm

A recipe search platform with a custom-built search algorithm, benchmarked for performance, as part of a front-end development training program.

🔗 **[Live Demo](https://gwen-herissant.github.io/GwenaelleHerissant_7_23062022/)**

## Project Overview

This project was completed as part of a structured front-end development curriculum. The challenge was to design and implement an efficient search algorithm for a recipe platform, capable of filtering through a JSON dataset in real time.

The project went beyond just writing code — it involved **analyzing and comparing two algorithm implementations**, benchmarking their performance, and documenting the reasoning behind the final choice. Green Code principles were also taken into account throughout the process.

## Objectives

- Design a user-friendly search interface using a front-end framework
- Develop two versions of a search algorithm and filter recipes from a JSON dataset
- Benchmark and compare algorithm performance
- Document the approach and justify the choice of the optimal solution
- Apply Green Code principles to minimize unnecessary resource usage

## Key Features

- **Real-time search** — filters recipes dynamically as the user types
- **Tag-based filtering** — combine search keywords with ingredient, appliance, and utensil filters
- **Algorithm benchmarking** — two approaches implemented and compared using performance testing tools (Jsben.ch)
- **Green Code** — algorithm designed with efficiency and resource consumption in mind

This project is part of the **OpenClassrooms Front-End Developer** curriculum. It stands out from the others by introducing algorithmic thinking and performance analysis as core developer skills, alongside front-end implementation.
